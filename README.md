# MicaOs
MicaOs is a lightweight, custom web-based operating system designed to offer a fast, clean, and minimalist user experience. Built with HTML and NW.js, it runs efficiently as an optimised workspace that can be accessed directly inside your web browser or as a desktop environment.

           WELCOME TO MICA OS
Version: lithos
Creator: @fastkash2012
Built with: HTML & NW.js

--- ABOUT MICA OS --- 
MicaOs is a custom web-based operating system that is incredibly lightweight (30kb) and features a clean, minimalist design.
It is built for people who want a simple operating system with no bloatware and a fully customizable experience. Built on HTML,
it uses web-based applications and tools. It is very easy to develop apps and tools for, and it even includes an 
intuitive studio tool for creating theme packs, applications, and content.

--- SETUP INSTRUCTIONS --- 
1. Run the 'MicaOs.exe' file to launch the OS. 
   (Note: Windows might show a blue "Windows protected your PC" warning because this is an indie app. Just click "More info" -> "Run anyway".)

2. Install the Browser:
   - Navigate to the Start Menu inside MicaOs.
   - Select "Install App".
   - Locate the included 'google.mapp' file and install it.

3. Restoring Old Data:
   - If you have previously used this OS, import your '.mica' backup file to continue right where you left off.

--- MICA OS FILE TYPES --- 
This OS uses a custom file system. Here is what they do:
* .mapp  = Applications
* .mpkg  = Theme Packs
* .mica  = OS Backups
* .webm  = Videos (It accepts standard .mp4 files; just rename the file extension to .webm)
* .mupdate = for firmware updates and update packs
